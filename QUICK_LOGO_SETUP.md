# Quick Logo Setup Guide

## Option 1: Use the Helper Script (Easiest)

1. Open PowerShell in the project folder
2. Run: `.\add-logo.ps1`
3. When prompted, paste the full path to your logo image file
4. The script will copy it to the correct location

## Option 2: Manual Setup

1. **Find your logo image file** on your computer
2. **Copy the file** to this location:
   ```
   C:\Users\Galal\OneDrive\Desktop\Joseph_s website\public\logo.png
   ```
3. **Rename it** to `logo.png` (or `logo.jpg` if it's a JPG file)
4. **Refresh your browser** (Ctrl+Shift+R)

## Supported Image Formats
- PNG (recommended)
- JPG/JPEG
- WebP

## File Requirements
- Square aspect ratio (1:1) works best
- Recommended size: 200x200px or larger
- The image will be automatically cropped to a circle

## After Adding
- The logo will appear in:
  - Header (centered)
  - Home page hero section
  - Admin login page
- If it doesn't show, hard refresh: **Ctrl+Shift+R**
